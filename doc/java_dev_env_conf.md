#### 2. Window 系统

- 进入环境配置

![](../images/1600589320469-2694c4f6-0f19-421b-828a-acea887e6ded.png)

- 配置JAVA_HOME

```
D:\software\openjdk\java-se-8u41-ri
```

- 配置 CLASSPATH

```
.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar
```

- 添加到path 中

```
%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin
```
