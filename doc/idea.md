- MyBatisX : 方便 Mapper 接口与 Xml 文件之间的导航。

- Alibaba Java Coding Guidelines 阿里巴巴Java开发规范
- Lombok 代码简化工具
- Key Promoter X快捷键学习工具
- ReadHub 每天高效浏览行业资讯
- CodeGlance 代码地图插件
